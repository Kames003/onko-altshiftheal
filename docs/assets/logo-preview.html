Logo system pro digitálního průvodce onkologického pacienta.
Tři koncepty — klikni na zvolený, abys označil výběr.

Koncepty — Výběr směru
A — Rising Path
Tri kroky stúpajúce diagonálne — priamo metafora liečebnej cesty. Väčší bod = cieľ. Halo ring = dosažiteľnosť.
Odporúčané
B — Compass Guide
Kompas ako symbol navigácie a vedenia. "Průvodce" = sprievodca. Klasická medicínska orientácia — severne = vpred.
C — Forward Arrow
Čistá šípka nahor. Minimalistická, silná, okamžite čitateľná. Vidlicový chvost = "odkiaľkoľvek, smer je vopred".
